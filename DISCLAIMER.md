# Disclaimer

This is a fan project and isn't affiliated with or endorsed by Valve.

Nearly all the cursor artwork in this repository belongs to Valve Corporation. It
was taken from Dota 2's game files and converted into desktop cursor themes for
personal use on Linux and Windows. Nobody here claims to own it, and it isn't
offered under any license. If you're from Valve and want it removed, open an issue
and it comes down.

These are desktop cursors and nothing more. They don't touch Dota, they don't
unlock or alter anything in the game, and they aren't a substitute for the cursor
packs Valve sells. If you want the packs in Dota, buy them in Dota.

The exception is the text caret. Dota ships no text cursor, so the I-beam is drawn
from scratch and tinted from each pack's own colors, which makes it the one piece
of artwork here that's mine and not Valve's. That caret, the install script, the
Windows `install.inf` files, and the release workflow are all my own work,
released into the public domain. Do whatever you want with them.
